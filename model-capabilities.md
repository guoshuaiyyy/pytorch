# DeepSeek v4 Pro — 模型与能力说明

## 基础信息

- **模型名称**: DeepSeek v4 Pro
- **运行框架**: Anthropic Claude Code (VSCode 扩展)
- **平台环境**: Windows 11 / PowerShell 5.1

## 代码工程能力

### 文件操作
- 精确读取任意文件（支持代码、图片、PDF、Jupyter Notebook）
- 基于字符串精确匹配的文件编辑（Edit）
- 整文件写入（Write）
- 支持 Glob 模式批量匹配文件路径

### 代码搜索
- 基于 ripgrep 的正则表达式全文搜索
- 支持按文件类型、glob 模式过滤
- 支持多行匹配模式

### 终端与命令
- PowerShell 5.1 命令执行
- Git 操作（commit、branch、rebase 等）
- 包管理、构建、测试运行
- 支持后台任务异步执行
- 最长 10 分钟超时

## 项目协作能力

### 任务规划
- Plan Mode：复杂任务先规划、用户审批后执行
- TodoWrite：多步骤任务列表实时跟踪
- 多 Agent 并行：独立子任务可并发委托给子 Agent

### 记忆系统
- **用户记忆**: 记录用户角色、偏好、知识背景
- **项目记忆**: 记录项目目标、约束、关键决策
- **反馈记忆**: 记录工作方式偏好（该做/不该做）
- **参考记忆**: 记录外部系统资源位置
- 跨会话持久化，自动关联和回溯

### 定时任务
- CronCreate：定期或一次性定时提醒
- ScheduleWakeup：循环模式下的自定节奏唤醒

## 外部交互能力

- **WebSearch**: 实时网络搜索（美国地区可用）
- **WebFetch**: 获取网页内容并提取信息
- **Skill 技能系统**: 可加载专用技能扩展能力
  - 配置更新、键盘绑定、代码审查
  - PR Review、安全审查
  - 应用启动验证、模型 API 开发

## 安全与权限

- 分层的权限控制机制
- 破坏性操作需要用户确认
- 默认拒绝不安全操作
- 遵循 OWASP 等安全最佳实践

## 限制与说明

- PowerShell 5.1 不支持 `&&`、`||`、三元运算符等 Bash 语法
- 无内置 Docker/NPM/Node 运行时（需系统安装）
- 命令执行有 10 分钟超时上限
- WebFetch 不支持需要认证的页面
